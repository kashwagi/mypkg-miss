## 概要
ROS2環境下においてtalker及びlistenerを用いて通信処理を行い、ROS及びロボットシステムへの理解を深めることを目的としたパッケージです。
ROS2 Humbleでの使用を想定しています。

## 開発環境
ubunntu 22.04.1LTS

Python 3.10

ROS2 Humble

## ノード及びトピックの概要

### ノード

#### talker
実行後変数を1ずつ増やし続ける。

#### listener
受け取ったトピックの内容をを標準出力するためのもの。

### トピック

#### countup
talkerの変数をlistenerに送るためのもの。
